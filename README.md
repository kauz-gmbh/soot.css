# soot.css

A classless CSS library for drop-in brutalist design. Dark, bold, and minimalistic.

## Installation

### npm

```bash
npm i soot.css
```

### CDN

```html
<link rel="stylesheet" href="https://unpkg.com/soot.css">
```
