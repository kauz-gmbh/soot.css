Usage

Link the stylesheet. Write HTML. Done.

<link rel="stylesheet" href="bs.css">

Override CSS variables to customize.

Configuration

CSS variables. Override as needed.

Variable Default Description

--color-muted #a3a3a3 Muted text, captions

--line-height-base 1.6 Body copy line height
--line-height-tight 1.25 Heading line height
--line-height-relaxed 1.7 Long-form line height
--letter-spacing-tight -0.025em Heading tracking
--letter-spacing-wide 0.05em Uppercase labels
--border-thin 1px solid rgba(245, 245, 245, 0.1) Inline dividers
--border-normal 2px solid rgba(245, 245, 245, 0.1) Sections, tables, forms
--border-heavy 4px solid #f5f5f5 Blockquotes, callouts

Override

:root {
  --color-accent: #ff0066;
  --spacing-unit: 0.375rem;
  --max-content-width: 1200px;
}

Advanced Configuration

Full-bleed layout

Edge-to-edge content:

<body data-full-bleed>

Defaults

  • section + section get dividers
  • article stacks get lighter dividers
  • Content respects --max-content-width unless full-bleed
  • Print: light background, visible URLs, smart page breaks

Features

  • Zero classes
  • Dark theme, high contrast
  • No animations
  • System fonts
  • Responsive
  • Accessible defaults
  • Print-ready
